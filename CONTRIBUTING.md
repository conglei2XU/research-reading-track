# Contributing Guide

本仓库采用轻量维护原则。提交内容时，优先保证信息清楚、来源可追溯、状态可判断。

## 1. 提交前检查

提交前请确认：

- 论文条目包含 DOI、URL 或稳定来源；
- release log 包含项目链接、版本号或发布日期；
- 阅读任务包含负责人和状态；
- 阅读笔记不是简单复制摘要，而是包含最基本的判断；
- 不上传出版社版权 PDF、未脱敏资料或敏感内部文件。

## 2. 推荐提交方式

小更新可以直接提交到对应 Markdown 文件。  
较重要的结构调整、模板修改或批量更新建议通过 Pull Request 提交。

## 3. Commit message 建议

```text
Add 2026-W27 reading list
Update 2026-W27 release logs
Add paper note for AuthorYear_ShortTitle
Fix reading status labels
```

## 4. 最低内容标准

### 论文阅读条目

至少包含：

- Title
- Authors
- Year
- Source
- DOI / URL
- Assigned To
- Status
- Notes

### Release log 条目

至少包含：

- Project / Tool name
- Version / Release date
- URL
- Key changes
- Relevance to our work
- Status
