# Status and Naming Rules

## 1. Reading Status

- `To read`：待阅读
- `Reading`：阅读中
- `Done`：已完成
- `Discussed`：已在组会讨论
- `Use in review`：可纳入综述或理论框架
- `Exclude`：不纳入核心材料

## 2. Release Status

- `New`：新发现的 release
- `Tracking`：需要继续关注
- `Reviewed`：已查看并记录
- `Important`：对团队研究或实验有明显影响
- `No action`：已查看，但暂不需要处理
- `Deprecated`：不再继续关注

## 3. File Naming

### Weekly reading list

```text
YYYY-W##.md
```

Example:

```text
2026-W27.md
```

### Weekly release log

```text
YYYY-W##.md
```

Example:

```text
2026-W27.md
```

### Weekly update summary

```text
YYYY-W##.md
```

Example:

```text
2026-W27.md
```

### Paper note

```text
AuthorYear_ShortTitle.md
```

Example:

```text
Ostrom1990_GoverningCommons.md
```
